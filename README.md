# SeuPiM
Simple &amp; Easy-to-Use Project &amp; Issue Management Tool

個人・小規模チーム向けに「シンプルで使いやすい」を追求したプロジェクト管理ツールです  
  
SeuPiM は Redmine や Jira、Trac など、有名なプロジェクト管理ツールのように  
チケット（≒タスク）単位でプロジェクトを管理します  
  
複雑な設定やサーバー構築は不要、インストール後すぐにご利用頂けます  
  
こんな方におすすめしています  
  ・Excel でタスク管理をしている  
  ・Redmine を構築するにはサーバーの用意、社内手続きが大変で時間がかかる  
  ・小規模案件を手軽に管理したい  
  ・プロジェクト状況を見える化したい  
  ・個人開発 や 副業案件を管理したい  
  
「Excel管理では物足りないが、Redmineは大げさ」という方、是非お試しください  
  
  
PiiM は以下の機能を提供します  
  ・プロジェクト管理  
  ・タスク管理  
  ・進捗率管理  
  ・ガントチャート表示  
  ・タスク検索・フィルタリング  
  ・並び替え機能  
  ・ブラウザから利用可能  
  ・ローカル環境で完結
